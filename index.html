Second paragraph

  • :nth-child CSS Selector
  • nth-child(2) - Styles the 2nd child only
  • nth-child(odd) - Styles 1st, 3rd, 5th... (odd items)
  • nth-child(even) - Styles 2nd, 4th, 6th... (even items)
  • nth-child(3n) - Styles every 3rd item (3rd, 6th, 9th)
  • nth-child(3n + 1) - Styles every 3rd element starting from the 1st one
